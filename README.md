# Модуль анализа требований по модели Кано

Основной модуль `kano_analysis.py` реализует процессы, описанные в техническом задании:

- создание структуры опроса по модели Кано;
- формирование функциональных и дисфункциональных вопросов для каждого свойства продукта;
- импорт и анализ ответов респондентов;
- автоматическая классификация требований по категориям Кано;
- расчет коэффициентов удовлетворенности и неудовлетворенности;
- генерация интерпретации и рекомендаций.

## Структура модуля

- `KanoSurvey` — добавление свойств продукта и генерация опроса;
- `KanoAnalyzer` — обработка ответов, классификация и формирование отчета;
- `KanoFeatureResult` — результат анализа по каждому свойству.

## Пример использования

```python
from kano_analysis import KanoSurvey, KanoAnalyzer, SurveyResponse

survey = KanoSurvey()
survey.add_feature("F1", "Удобный интерфейс")
survey.add_feature("F2", "Автоматическая сортировка требований")

responses: list[SurveyResponse] = [
    {"F1": (1, 4), "F2": (3, 2)},
    {"F1": (2, 4), "F2": (4, 1)},
    {"F1": (1, 5), "F2": (3, 3)},
]

analyzer = KanoAnalyzer()
results = analyzer.analyze(survey, responses)
print(analyzer.generate_report(results))
```

## Ввод ответов через консоль

Запустите модуль напрямую:

```powershell
python kano_analysis.py
```

После запуска система предложит:

1. ввести имя файла опросника (созданного в `create_survey.py`);
2. ввести количество респондентов;
3. для каждого свойства задать функциональный ответ;
4. для каждого свойства задать дисфункциональный ответ.

По завершении будет сгенерирован полный отчет по анализу Кано и создан график распределения ответов.

## График распределения

После анализа автоматически создается столбчатый график `kano_chart.png` с распределением всех ответов по категориям Кано (A, O, M, I, R, Q).

Требуется установка matplotlib:

```bash
pip install matplotlib
```

Если matplotlib не установлен, график не создается, но анализ работает.
## Создание и управление опросниками

Для создания реюзабельных опросников используйте скрипт `create_survey.py`:

```powershell
python create_survey.py
```

Скрипт позволяет:

- **Создать новый опросник**: ввести свойства продукта через консоль, сгенерировать вопросы по шаблону Кано и сохранить в JSON-файл.
- **Загрузить существующий**: открыть сохраненный опросник, просмотреть вопросы и при необходимости добавить новые свойства.

### Формат сохранения

Опросники сохраняются в JSON-файлы для повторного использования. Пример структуры:

```json
{
  "features": [
    {
      "feature_id": "F1",
      "name": "Удобный интерфейс",
      "description": "Интуитивно понятный пользовательский интерфейс"
    }
  ]
}
```

## Формат ответов

Ответы могут быть числами от `1` до `5` или текстовыми значениями:

- `1` — Мне нравится
- `2` — Ожидаю
- `3` — Нейтрально
- `4` — Могу терпеть
- `5` — Мне не нравится
